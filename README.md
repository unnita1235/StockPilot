**Elevator pitch:**  
A real-time inventory management platform that reduces stockouts by 78% for retailers using predictive analytics.

## Problem
Retailers lose millions annually due to inaccurate inventory tracking and stockouts.

## Solution Overview
- Real-time inventory dashboard (React + Next.js)
- Predictive analytics for stockout prevention (Node.js + MongoDB)
- Role-based authentication (JWT)
- Automated reporting and alerts

## My Role
- Designed and implemented full-stack architecture (Next.js + NestJS)
- Developed REST API and MongoDB models
- Integrated CI/CD, Docker, and secret scanning

## Quick Start
```bash
git clone https://github.com/unnita1235/StockPilot.git
cd StockPilot

# Start backend
cd backend && npm install && npm run start:dev

# In another terminal, start frontend (from root)
cd StockPilot && npm install && npm run dev
```
